# SmartGestex
SmartGestex is a touchless human–computer interaction system that enables control of mouse, presentations, and applications using hand gestures and voice commands.

---

## 📌 Overview
SmartGestex is a touchless human–computer interaction system that allows users to control computer operations using hand gestures and voice commands. The system eliminates the need for traditional input devices like a mouse and keyboard.

---

## 🚀 Features
- Gesture-based mouse control  
- Slide control for presentations  
- Air drawing using hand gestures  
- Voice command system  
- User authentication (username & password)  
- Real-time interaction  

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- MediaPipe  
- PyAutoGUI  
- SpeechRecognition  
- Tkinter  

---

## 🎯 Applications
- Smart classrooms (hands-free teaching)  
- Assistive technology for disabled users  
- Healthcare (touchless systems)  
- Public interactive systems  

---

## ⭐ Advantages
- No physical contact required  
- Easy to use
